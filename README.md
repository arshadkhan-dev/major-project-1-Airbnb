  # Major Project 1 - Airbnb Clone

An Airbnb-inspired full-stack web application where users can browse, create, edit, and manage property listings.

Built using HTML, CSS, JavaScript, EJS, Node.js, Express.js, MongoDB, and Mongoose.


## Features

- Create, edit, and delete property listings
- View detailed listing pages
- Image upload support
- Responsive UI design
- MongoDB database integration
- RESTful routing
- Server-side rendering using EJS
